# justmarkup.com


The 2019 version of justmarkup.com based on [Eleventy Base Blog](https://github.com/11ty/eleventy-base-blog/)


## Run Eleventy

```
npx eleventy
```

Or build and host locally for local development
```
npx eleventy --serve
```

Or build automatically when a template changes:
```
npx eleventy --watch
```

Or in debug mode:
```
DEBUG=* npx eleventy
